# nyaa

## setup
```bash
~/nyaa $ python3.6 -m venv venv
~/nyaa $ venv/bin/pip install -r requrements.txt
```

## run
```bash
~/nyaa $ venv/bin/python3 nyaa.py -h
usage: Anime nyaa [-h] [--torrent_dir TORRENT_DIR] [--db_path DB_PATH]
                  [--db_file DB_FILE] [--add_rss ADD_RSS] [--del_rss DEL_RSS]
                  [--add_anime ADD_ANIME [ADD_ANIME ...]]
                  [--del_anime DEL_ANIME] [--info]

optional arguments:
  -h, --help            show this help message and exit
  --torrent_dir TORRENT_DIR
  --db_path DB_PATH
  --db_file DB_FILE
  --add_rss ADD_RSS
  --del_rss DEL_RSS
  --add_anime ADD_ANIME [ADD_ANIME ...]
  --del_anime DEL_ANIME
  --info, -i

```
